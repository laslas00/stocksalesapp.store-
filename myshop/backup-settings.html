Backup System Tutorial

1 Welcome to StockApp* Backup System

This tutorial will guide you through the backup and restore features to protect your valuable inventory data.

Your data is automatically protected with encrypted backups

The backup system helps you:

  • Protect against data loss
  • Recover from accidental deletions
  • Migrate data between computers
  • Schedule automatic backups

2 Automatic Flash Drive Backup

StockApp* can automatically backup your data to connected USB flash drives.

How it works:

  • prepare a unique USB flash drive for this aplicatin and name it stcokapp drive
  • then Insert the USB flash drive
  • and wait for the System automatically detects it
  • Creates encrypted backups automatically
  • Keeps last 30 backups (configurable)

Backup Frequency: You can configure backups to run:

  • Every hour
  • Every 6 hours
  • Every 12 hours (default)
  • Daily or weekly

3 Manual Backup & Restore

You can manually create backups or restore from existing ones at any time.

Manual backup gives you control over when to backup

Important Notes:

  • Backups are encrypted with AES-256
  • Password: StockApp*0012
  • Backup format: .zip files
  • Includes all data: stock, sales, users, settings

4 How to Restore Data

Restoring is simple and can be done in two ways:

From Backup List

Select from detected backups on connected flash drives

Manual File

Browse and select any .zip backup file from your computer

Restore Process:

  1. Select backup file
  2. System verifies the backup
  3. Creates backup of current data (safety)
  4. Restores selected backup
  5. Restarts application

5 Restore During Setup

If you're setting up a new installation, you can restore from an existing backup.

Try it now:

Click the "Restore from Backup" button below to:

  • Upload your backup .zip file
  • Restore all previous data
  • Skip manual setup process

Tip: Restore is perfect for:

  • Migrating to a new computer
  • Recovering from system crash
  • Setting up on additional devices
